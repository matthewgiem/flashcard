Flashcard

Javascript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

An operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

JavaScript variables are containers for storing data values.

Variable naming conventions

Variables should begin with a letter. Variables are case sensitive (myNumber is a different variable than myNUMBER). Use clear names that describe the value being stored like myNumber.

Functions

A function is a block of code that performs an action and returns a result; optionally takes arguments

Methods

method: A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

argument: Some methods take one or more arguments that provide the method with additional information to help it perform its action.

Parameters

The argument object contains an array of the arguments used when the function was called.

Return

return value: The return value is the method's response.

Chaining methods

Calling a method directly on the return value of another method.

Strings

A JavaScript data type that represents the exact text of whatever is enclosed in the quotes

Booleans

A boolean operator is a JavaScript data type that is either false or true.

Undefined

The undefined property indicates that a variable has not been assigned a value.

Escape

Escape is a function in javascript that encodes a string making it portable across any network.

NaN

Not A Number.

Alert

Method in JavaScript that displays a message.

Comments

Comment: anything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.

Attributes

Attributes provide additional information about HTML elements and provide additional information about an element

jQuery

A library of javascript code preforming basic tasks
